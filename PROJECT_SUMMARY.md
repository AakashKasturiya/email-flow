# 📖 Overview

MailFlow is a full-stack email management platform that allows users to create professional emails using reusable templates, attach files, preview emails before sending, and monitor email delivery statistics through an interactive dashboard.

The application is designed for businesses, marketers, HR teams, and developers who need an efficient way to manage email communication.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/mailflow.git
```

```bash
cd mailflow
```

---

## Install Frontend

```bash
cd frontend
npm install
```

Run

```bash
npm run dev
```

---

## Install Backend

```bash
cd backend
npm install
```

Run

```bash
npm run dev
```
